# JAVA PROJECTS
This repository contains three Java projects: GradeCalculator, NumberGuessing, and CurrencyCalculator.

1. _NumberGuessing Game_
Description
The NumberGuessing project is a simple number guessing game where the user has to guess a randomly generated number within a given range. The game tracks the number of attempts and provides feedback after each guess.

**Features**
Generates a random number between a given range (e.g., 1-100).
Allows the user to guess the number.
Provides feedback on whether the guess is too high, too low, or correct.
Tracks the number of attempts.
Limits the number of attempts (optional feature).

Example:
Guess the number between 1 and 100: 50
Your guess is too high.

Guess the number between 1 and 100: 30
Your guess is too low.

Guess the number between 1 and 100: 40
Congratulations! You've guessed the number correctly in 3 attempts.

***How to Run***
Clone the repository.
Navigate to the Task-1 folder.
Run the EnhancedGuessingGame.java file.

2. _GradeCalculator_
Description
The GradeCalculator project calculates the grade of a student based on their marks obtained in different subjects. It takes the marks (out of 100) for each subject, calculates the total marks, average percentage, and assigns a grade based on the percentage.

**Features**
Takes marks as input for each subject.
Calculates total marks and average percentage.
Assigns a grade based on the average percentage:
A for 90% and above
B for 80% to 89%
C for 70% to 79%
D for 60% to 69%
F for below 60%

Example:
Marks for Subject 1: 80
Marks for Subject 2: 85
Marks for Subject 3: 78
Total Marks: 243
Average Percentage: 81.0%
Grade: B

**How to Run**
Clone the repository.
Navigate to the Task-2 folder.
Go to the src folder.
Run the GradeCalculator.java file.

3. _CurrencyCalculator_
Description
The CurrencyCalculator project converts an amount from one currency to another based on a user-defined exchange rate. It supports various currencies and allows users to input the amount to be converted.

**Features**
Takes user input for the amount to be converted.
Allows the user to choose the source and target currency.
Displays the converted amount based on the exchange rate.
Supports multiple currencies (e.g., USD, EUR, INR).

Example:
Enter amount in USD: 100
Convert from: USD
Convert to: EUR
Converted Amount: 85.0 EUR

**How to Run**
Clone the repository.
Navigate to the Task-3 folder.
Go to the src folder.
Run the CurrencyConverter.java file.
